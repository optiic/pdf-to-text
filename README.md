### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/optiic/pdf-to-text.git
cd pdf-to-text
npm install
npm start
```

## ✨ Features

- Extract text from any PDF document
- OCR support for scanned and image-based PDFs
- Batch processing via REST API
